## Common Workflows

### Take a Screenshot
- Full screen: `⌘ + ⇧ + 3`
- Selected area: `⌘ + ⇧ + 4`
- Specific window: `⌘ + ⇧ + 4`, then `Space` + click

### Switch Between Apps and Windows
- Between apps: `⌘ + Tab`
- Between windows of the same app: `` ⌘ + ` ``

### Find Files and Launch Apps
- `⌘ + Space` — Spotlight (search files, apps, web)
- `⌘ + ⇧ + G` — Go to specific folder path (in Finder / dialogs)

### Force Quit a Frozen App
1. `⌘ + ⌥ + Esc` — Open Force Quit dialog
2. Select the frozen app and click Force Quit

### Lock Screen
- `⌃ + ⌘ + Q` — Lock screen immediately

## Key Differences: Mac vs Windows

| Behavior | Mac | Windows |
|----------|-----|---------|
| Primary modifier for shortcuts | `⌘ Command` | `Ctrl` |
| Closing a window | `⌘ + W` (app stays open) | `Ctrl + W` or `Alt + F4` |
| Quitting an app | `⌘ + Q` | `Alt + F4` |
| Task Manager | `⌘ + ⌥ + Esc` | `Ctrl + Alt + Del` |
| Home / End keys | `⌘ + Left/Right Arrow` | `Home` / `End` key |
| Delete (backspace) | `Delete` key | `Backspace` key |
| Forward delete | `fn + Delete` | `Delete` key |
| Right-click | `⌃ + click` or two-finger tap | Right mouse button |
| Maximize window | Green button or double-click title bar | Maximize button (full screen) |
| Screen search/launcher | `⌘ + Space` (Spotlight) | Windows key |
| Emoji picker | `⌃ + ⌘ + Space` | `Win + .` |
| Lock screen | `⌃ + ⌘ + Q` | `Win + L` |
| Screenshot | `⌘ + ⇧ + 3/4/5` | `Print Screen` / `Win + ⇧ + S` |
