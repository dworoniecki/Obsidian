## Common Workflows

### Create and Format a Table
1. Select data range
2. `‚åò + T` ‚Äî Create Table
3. `‚åò + ‚áß + L` ‚Äî Toggle Filters
4. `‚åò + 1` ‚Äî Format Cells dialog

### Copy Values Only (Remove Formulas)
1. Select the cell range
2. `‚åò + C` ‚Äî Copy
3. Navigate to destination
4. `‚åò + ‚å• + ‚áß + V` ‚Äî Paste Values Only

### Flash Fill a Column
1. Type the desired output in the first cell
2. `‚åò + E` ‚Äî Flash Fill (auto-detects pattern)

### Insert and Delete Rows/Columns
- `‚åò + ‚áß + =` ‚Äî Insert row or column
- `‚åò + -` ‚Äî Delete row or column

### Quick Navigation to Data Edges
- `‚åò + Arrow Key` ‚Äî Jump to edge of data region
- `‚åò + ‚áß + Arrow Key` ‚Äî Select to edge of data region
